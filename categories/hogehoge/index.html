ブログをgithubで開設

 May 5, 2017
Hello,Github.io!! ようやっとのことでgithub.io+Hugoを使ってブログ開設できたのでその紹介を。 まずgithub.ioについて。 Githubにはバージョン管理システムの他にも、静的なサイトだけ表示してくれるGithub pagesというサービスがあります。<自分のアカウント名リポジトリ>.github.ioというリポジトリにhtmlをpushすれば自前でサーバーを持たずとも表示してくれるシステムです。 Hugoとはなにか？ 簡単に言うと、ホームページ生成ジェネレータの一種です。こういった類のものは他にもありますが、生成の速度が他より早いことが利点として挙げられます。また、テーマも豊富に揃っているため、好きなものも選べます。(本ページのテーマ: hugo-tranquilpeak-theme) 詰まったところ Hugoディレクトリ以外でHugoコマンドを認識するにはパスを通す必要がある gitのコマンドがいまいちわかっていなかったこと←最重要 各種パラメータ調整 etc… 注意すべき点 Hugo ではローカルサーバーでプレビュー表示ができるが、そのときに > hugo server 使うテーマ名 -D -w でないと相対パスが崩れて参照できなくなるようだ。 Github Pagesはhttpではなくhttpsである。 作ったきっかけ 知見共有としてのメモをどこかに置いておきたい＋サーバーを借りないでサイトができること。 続くといいなぁ。 参考にしていただいたサイト http://tanksuzuki.com/post/hugo-github-pages-1/ https://myaox.github.io/post/InstallHugo/
